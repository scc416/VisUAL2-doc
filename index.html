VisUAL2: ARM Simulator

Model, View, Update

VisUAL2 assembles and simulates the ARMv4T instruction sets with condition field. The application follows the pattern of MVU:

Model

Immutable data structure that contains the state of the application
Function init produces the initial model

View

Pure function that produces UI (React elements) with the current model. When user interacts with the UI, a message is sent to “Update”

Update

Pure function that produces the new model with the message and previous model.
